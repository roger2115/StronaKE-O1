# SCiPnet | Orientacja KE

Interaktywny portal Fundacji SCP zawierający informacje o **Komitecie ds. Etyki** oraz **Mobilnej Formacji Operacyjnej Omega-1**.

## 🎯 O Projekcie

Portal zapewnia dostęp do:
- Hierarchii i struktury Komitetu ds. Etyki
- Regulacji dotyczących personelu Klasy D
- Informacji o jednostce Omega-1 i procedurach zgłaszania
- Instruktażu dotyczącego protokołów interwencji etycznej

## 🚀 Funkcje

- **Responsywny design** - działa na Desktop, Tablet i Mobile
- **Animowane elementy** - loga z efektami rotacji i glow
- **Tabele regulacji** - pełne wytyczne Klasy D
- **Hierarchie organizacyjne** - struktura KE i Omega-1
- **Nowoczesny UI** - glass-morphism i gradient design

## 📁 Struktura Projektu

```
StronaKE-O1/
├── index.html          # Główny plik HTML
├── styles.css          # Stylowanie CSS
├── script.js           # Logika JavaScript
├── modules/            # Moduły JS
│   ├── ui.js          # Nawigacja
│   ├── terminal.js    # Efekty terminala
│   └── regulations.js # Wytyczne Klasy D
├── Assets/            # Grafiki i loga
└── README.md          # Dokumentacja
```

## 🛠️ Technologia

- **HTML5** - semantyczna struktura
- **CSS3** - gradient, animation, glow effects
- **Vanilla JavaScript** - bez frameworków
- **GitHub Pages** - hosting
