  └─ Assets/              ← Zdjęcia

---

## 🛠️ Jeśli Coś Nie Działa

### Loga się nie pokazują?
→ Upewnij się że folder `Assets/` jest w tym samym folderze co `index.html`

### Skrypty się nie ładują?
→ Sprawdź konsolę (F12) i upewnij się że `modules/` folder istnieje

### Strona wygląda dziwnie?
→ Odśwież przeglądarkę (Ctrl+F5) aby wyczyścić cache

## 💡 Pro Tips

1. **F12** → Otwórz Developer Tools
   - Sprawdź `Console` dla easter egg message
   - Sprawdź `Network` aby zobaczyć załadowane pliki
   - Sprawdź `Application` → `localStorage` (jeśli dodamy)

2. **Ctrl+Shift+M** (W Chrome)
   - Przełącz na Mobile View
   - Testuj responsywność

3. **Ctrl+F5**
   - Hard refresh (wyczyść cache)
   - Przydatne jeśli strona wygląda dziwnie

4. **Lighthouse** (W DevTools)
   - Uruchom Lighthouse audit
   - Sprawdź performance score

## 📞 FAQ

**P: Czy muszę mieć internet?**
O: Nie! Aplikacja działa całkowicie offline. (Wytyki ładują się z fallback content)

**P: Czy to bezpiecze?**
O: Tak! Żaden kod nie jest wysyłany na serwer. Wszystko dzieje się lokalnie.

**P: Czy mogę to edytować?**
O: Oczywiście! Plik HTML, CSS i JS są całkowicie dostępne do modyfikacji.

**P: Czy to będzie działać w innej przeglądarce?**
O: Tak! Testowano na Chrome, Firefox, Safari, Edge.
