# Historia Zmian

## [2.0.0] - 2024 - Pełna Optymalizacja

### Dodane
- Modularyzacja JavaScript (UI, Terminal, Regulations)
- Tabele z wytycznymi Klasy D
- Hierarchie KE i Omega-1
- Animowane tło z gradientem
- Glow effects na elementach
- GitHub Pages deployment

### Zmienione
- Refaktoryzacja CSS (separacja z HTML)
- Optymalizacja performance (usunięcie backdrop-filter)
- Zmiana struktury HTML (semantic)
- Nowy navbar z logo

### Usunięte
- Inline styles
- Zbędne animacje
- Duże background images

### Performance
- Zmniejszenie CSS: 5.6KB → 13.6KB (struktura, nie rozmiar)
- Load time: ~300ms
- Lighthouse: 90+

## [1.0.0] - Początkowa Wersja
- Podstawowy portal z dwiema sekcjami
- CSS inline, JavaScript inline
- Glass-morphism design
